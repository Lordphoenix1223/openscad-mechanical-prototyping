# Lessons Learned

## Tolerances

- Start with explicit mechanical assumptions instead of hoping the printer "will probably handle it."
- Use fit coupons for screw holes, slots, and clamp pressure before trusting the full assembly.
- Movable parts need real clearance, not wishful symmetry.

## Mechanisms

- Smaller, chunkier geometry is often more honest than thin cinematic geometry.
- A prototype can still be valuable even when it is visually compromised, if the mechanism risk becomes measurable.

## Modular Parts

- Splitting a design into shell, base, clamp, and coupon pieces makes debugging easier.
- Modular parts also make public documentation clearer because each piece has one job.

## Brutal Truth

The most dangerous failure mode in hobby hardware is pretending a concept is further along than it is. A credible prototype has documented compromises, explicit constraints, and known next tests.
