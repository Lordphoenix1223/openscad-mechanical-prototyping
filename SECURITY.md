# Security Policy

This repository contains public CAD source, documentation, and render assets. It should not contain secrets, private service URLs, unpublished prompts, or machine-specific local paths.

## Reporting

- Prefer GitHub private vulnerability reporting if it is enabled for this repository.
- If private reporting is not available, open a minimal issue requesting contact and keep the report high level.
- Do not post secrets, private screenshots, or sensitive local environment details in issues or pull requests.

## Scope Notes

- There is no backend, auth layer, or provider integration in this repository.
- Security work here mostly means safe publication hygiene, documentation accuracy, and avoiding accidental disclosure.
